# 1.5 联合体

联合体类似 Rust 中的 `enum`，它表示一个不同情况下类型不同的值。

联合体是一个值类型。

定义它的语法如下：

```aoi
union UnionName {
    ConditionName0: Type0;
    ConditionName1: Type1;
    ...
}
```

其中，`UnionName` 表示联合体的类型名，`ConditionName` 表示情况名，`Type` 表示情况对应的类型。

用 `UnionName::ConditionName value` 创建一个对应情况的联合体值。

要获取联合体实例的情况和具体的值需要使用模式匹配，见后文。

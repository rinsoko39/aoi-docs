# 1.2 基础类型

Aoi 是强类型语言。在 Aoi 中有以下基础类型。

## 整数、浮点数、布尔、字符、字符串类型

Aoi 的数值、布尔、字符、字符串类型如下表所示：

类型 | 说明 |  字面量举例
| - | - | - |
`int` | 带符号 64 位整数 | `42` `0x42`
`float` | 64 位浮点数 | `4.2`
`bool` | 布尔值 | `true` `false`
`char` | UTF-16 字符 | `'a'` `'葵'`
`str` | UTF-16 字符串 | `"aoi"`

它们都是值类型。虽然 `str` 以引用存储，但它具有值类型的性质。

`0b` `0o` `0x` 前缀可用于表示不同进制的整数。

`0` 开头的数字并不会被转换成八进制。这太蠢了。

`\` 用于转义字符和字符串。

用 `` ` `` 包围的字符串可以用 `{}` 进行插值，并且支持多行。例如，`` `a = {a}` ``，当 `a` 值为 `42` 时，结果为 `"a = 42"`。

## `never` 类型

`never` 类型是一个特殊的类型，它表示该表达式的值永远不会得到，同时后续的代码也不会被执行。例如 `return` `break` `continue` 表达式均为 `never` 类型。

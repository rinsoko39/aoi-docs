# 1.3 枚举

枚举是一种自定义的基础值类型。

定义一个枚举的语法如下：

```aoi
enum EnumName {
    ValueName00;
    ValueName01
    ...
    ValueName10 = value;
    ValueName11;
    ...
}
```

其中，`EnumName` 是枚举的类型名，`ValueName` 是枚举值的名称，`value` 是枚举值对应的整数值。

枚举值以 64 位整数存储，没有指定整数值的枚举值默认从 `0` 开始依次向后排列。上例中，`ValueName00` `ValueName01` `ValueName10` `ValueName11` 的整数值分别为 `0` `1` `value` `value + 1`。

用 `EnumName::ValueName` 表示一个枚举值。

# 类型定义

每次都用 `dyn[...] ...` 表示动态类型比较繁琐，可以用类型定义简化：

```aoi
type TypeName[T0, T1, ...] = TypeDef
```

例如，`Iter[T, TVal]` 描述一个迭代器：

```aoi
trait Iter[T, TVal] {
    op iterate iter: T -> TVal?;
}
```

那么，动态类型 `dyn[T: ITer[T, TVal]] T` 描述一个迭代 `TVal` 类型的迭代器。可以通过类型定义简化它：

```aoi
type DIter[TVal] = dyn[T: Iter[T, TVal]] T;
```

建议定义动态类型时类型名以 `D` 开头。

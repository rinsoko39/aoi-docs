# 3.2 类型转换运算符

类型转换用于将一个类型转换为另一个类型。类型转换分为显式和隐式。

隐式转换无需标明。显式转换需要 `Type value` 将 `value` 转换为 `Type` 类型。

基础类型中，`int` 可以隐式转换为 `float`，`float` 可以显式转换为 `int`。

定义一个隐式转换的语法如下：

```aoi
convert implicit value: FromType -> ToType {
    ...
}
```

定义一个显式转换的语法如下：

```aoi
convert explicit value: FromType -> ToType {
    ...
}
```

其中，`value` 是一个赋值模式，待转换的值会赋给 `value`，`FromType` 和 `ToType` 是两个类型，分别为转换前和转换后的类型。`{ ... }` 是一个 `ToType` 类型的表达式块。

例如，以下代码定义了一个由自定义的整数向量到向量的隐式转换，以及反向的显式转换：

```aoi
struct Vector {
    x: float;
    y: float;
}
struct VectorInt {
    x: int;
    y: int;
}
convert implicit value: VectorInt -> Vector {
    new Vector {
        x: value.x,
        y: value.y
    }
}
convert explicit value: Vector -> VectorInt {
    new VectorInt {
        x: int value.x,
        y: int value.y
    }
}
```

之后，可以使用 `VectorInt a` 将 `Vector` 类型的变量 `a` 转换为 `VectorInt` 类型。

# 3.4 字面量后缀

可以为 `int` `float` `char` `string` 类型的字面量定义字面量后缀，将该类型的字面量转换为其它类型的值，以简化一些语法。对于名为 `suf` 的后缀，用 `42suf` `4.2suf` `'a'suf` `"aoi"suf` 的形式使用该后缀。后缀和字面量之间不能有空格。

定义字面量后缀的语法如下：

```aoi
literal value suffixName: ValueType -> ResultType {
    ...
}
```

其中，`suffixName` 是后缀名称，`value` 是声明模式，存储字面量的值，`ValueType` 是字面量类型，`ResultType` 是结果类型。`{ ... }` 是一个 `ResultType` 类型的表达式块。

例如，为自定义的复数类型添加字面量后缀 `i`，以简化语法：

```aoi
struct Complex {
    real: float;
    imag: float;
}
op a + b: Complex <> Complex -> Complex {
    new Complex {
        real: a.real + b.real,
        imag: a.imag + b.imag
    }
}
literal value i: float -> Complex {
    new Complex {
        real: 0,
        imag: value
    }
}
```

这样，就可以用类似 `4 + 2i` 的方法表示一个复数。
